# ProfileRadja

ProfileRadja adalah aplikasi Android sederhana yang dibuat menggunakan Jetpack Compose untuk menampilkan profil pengguna.

## Fitur

- Foto profil
- Nama pengguna
- Bio singkat
- Email
- Nomor telepon
- Lokasi
- Tombol Contact

## Teknologi

- Kotlin
- Jetpack Compose
- Material3
- Android Studio

## Cara Menjalankan

1.  Clone Repository
2.  Buka project di android studio
3.  Tunggu Grade Sync
4.  Klik tombol RUN
5.  Pilih emulator atau HP Android
